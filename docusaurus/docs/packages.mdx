# Packages

Use this page to inspect both the supported Slurm Factory build matrix and the live public Spack cache.
The cache browser runs in your browser and fetches current package manifests directly from the CloudFront-backed cache.

## Cache URL Patterns

| Artifact | URL pattern |
|----------|-------------|
| Slurm Spack mirror | `https://slurm-factory-spack-binary-cache.vantagecompute.ai/{toolchain}/slurm/{slurm_version}/` |
| Dependency Spack mirror | `https://slurm-factory-spack-binary-cache.vantagecompute.ai/{toolchain}/slurm/deps/` |
| Complete tarball | `https://slurm-factory-spack-binary-cache.vantagecompute.ai/{toolchain}/{slurm_version}/{architecture}/slurm-{slurm_version}-{toolchain}-{architecture}-software.tar.gz` |
| Tarball signature | `https://slurm-factory-spack-binary-cache.vantagecompute.ai/{toolchain}/{slurm_version}/{architecture}/slurm-{slurm_version}-{toolchain}-{architecture}-software.tar.gz.asc` |
